# Project created with rn.new / Create Expo Stack

## Current Expo SDK 52

## Requirements

- Node JS v20.18.x
- Android: Android Studio, Android SDK, Java
- iOS: xCode, Watcher, Cocoapod
- Expo account

## Installation

- Run `git clone git@github.com:mrlinnth/ces.git`
- Run `cd ces`
- Run `npm install`
- Run `npx expo-doctor` and follow instructions
